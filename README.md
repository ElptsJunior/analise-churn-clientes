# Análise de Churn de Clientes

## Objetivo
Analisar padrões de cancelamento de clientes para identificar fatores que contribuem para churn.

## Ferramentas

Python
Pandas
SQL
Power BI

## Dataset

Customer churn dataset

## Principais insights

Clientes com contrato mensal apresentam maior taxa de cancelamento.

Clientes com maior valor de mensalidade possuem maior probabilidade de churn.
